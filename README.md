# Simple ATM Simulation 
* This is a Core java-based Banking System project. It is a menu-driven application using switch case. <br>
* This project includes an OTP verification System where a random OTP is generated. <br>
* Thread.sleep() is used to provide a waiting time for OTP verfication. <br>
* Scanner class - To take User Input <br>
* try-catch - handling Exception <br>
* while loop - program keep running until user exit. <br>
# Operation like Deposite amount, Withdraw Amount, Check Amount.
